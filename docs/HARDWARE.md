# EDITH V3 Hardware

Core hardware:
- Raspberry Pi 4B
- Raspberry Pi Camera Module V2
- 1.51-inch transparent OLED display
- Mini USB microphone
- Bluetooth earbuds or AirPods
- Bluetooth/WiFi

Optional expansion:
- Elegoo Uno R3 for Iron Man suit/servo control
- Serial command bridge from Raspberry Pi to Uno R3

Current goal:
Build EDITH as a Raspberry Pi–based smart glasses AI assistant with voice, HUD, camera vision, Bluetooth device awareness, local/offline features, and optional cloud AI.
